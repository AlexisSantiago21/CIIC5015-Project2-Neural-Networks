# Proyecto II - Redes Neuronales Fully Connected
**Curso:** CIIC 5015 - Introducción a la IA
**Fecha de entrega:** 1 de abril de 2026

## Descripción
Este proyecto consiste en la implementación de modelos de aprendizaje profundo utilizando PyTorch 2.0 para tareas de:
1. **Regresión:** Predicción de MPG basado en atributos de autos.
2. **Clasificación:** Identificación de dígitos escritos a mano (MNIST).

## Requisitos
- PyTorch 2.0
- Jupyter Notebooks
- Google Colab (opcional)
